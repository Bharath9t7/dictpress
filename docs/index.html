Build dictionary websites for any language

DictPress is a lightweight, standalone, single binary webserver application for building and publishing fast, searchable dictionary websites for any language. Free and Open Source.

Live dictionaries

  • Alar: Kannada-English dictionary
  • Olam: English-Malayalam, Malayalam-Malayalam dictionary

How it works

DictPress is language agnostic and has no concept of language semantics. It stores all data in a Postgres database. To make a universal dictionary interface possible, it treats all dictionary entries as UTF-8 strings that can be searched with Postgres DB's fulltext capabilities by storing tsvector tokens alongside them. The tokens that encode and make the entries searchable can be anything—simple stemmed words or phonetic hashes like Metaphone.

Postgres comes with built-in tokenizers for two dozen languages (\dFd to see the full list on psql).

For languages that do not have Postgres fulltext dictionaries, search tokens can be generated externally and plugged in. For example, Olam uses MLPhone, a simple Metaphone like phonetic hashing algorithm that allows Malayalam words in the dictionary to be searched by how they sound.

Features

  • HTTP/JSON APIs for everything.
  • Support for custom themes and templates to publish dictionary websites.
  • Paginated A-Z (or any alphabet for any language) glossaries.
  • Admin dashboard for managing and curating dictionary data.
  • Accept and moderate user submissions (new dictionary entries and corrections) from the dictionary website.
  • Generic entry-relation-entry structure for dictionary data in a Postgres database with just two tables (entries, relations).
  • Works with any language. Use built in Postgres tokenizers or plug in external tokenizers.
  • Store dictionaries for any number of languages in a single database.
  • Extremely light weight and fast.

Why?

Dictionaries are key to making languages accessible to all. Every language should ideally have high quality, open source, easily accessible dictionaries and datasets. That few languages in the world have this, is sad.

DictPress provides an easy means to publish online dictionaries whether one is a student, linguist, university, or a government. The project hopes to see many more open dictionaries like Alar and Olam go online.
